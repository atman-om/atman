# Atman App Interface — v1.0.1

v1.0.1 adds `apps/atman`, a unified product shell that makes the system feel like one app instead of separate Studio/Public surfaces.

## URL

```text
http://localhost:3002
```

## Sections

```text
Dashboard
Ask Atman
Canonical Library
Corpus Intake
Qwen Serving
Content Factory
NyayaBench
Operations
```

## Positioning

- `apps/public` remains the clean public surface.
- `apps/studio` remains the admin power surface.
- `apps/atman` is the integrated command center/demo app for owners, operators, and product walkthroughs.
