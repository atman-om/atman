# Content Review Workflow

## Required checks

- Source chunks exist when source_required is true.
- No fake shloka or invented Sanskrit.
- Citations point to real chunks.
- Hindi is understandable for the target learner.
- The item does not claim universal doctrine when tradition differences matter.

## Decision meanings

| Decision | Meaning |
|---|---|
| APPROVED | Export/publish allowed |
| NEEDS_REVISION | Fix before export |
| REJECTED | Do not use |
| AUTO_BLOCKED | System-detected hard-risk |
